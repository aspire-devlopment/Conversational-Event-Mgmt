## Why These Packages Are Used

- `fastapi`
  HTTP API framework and schema-driven request validation.
- `uvicorn`
  ASGI server for local and production-compatible app hosting.
- `psycopg`
  PostgreSQL driver for direct SQL access.
- `PyJWT`
  JWT token encode/decode support.
- `httpx`
  Async HTTP client used for LLM provider calls.
- `pydantic-settings`
  Environment-driven application configuration.
- `alembic`
  Database migration support.
- `SQLAlchemy`
  Used by Alembic migration tooling.
- `dateparser`
  Natural-language date parsing for chat-driven event input.
